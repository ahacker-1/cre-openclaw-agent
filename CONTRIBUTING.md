# Contributing

Thanks for considering a contribution to CRE OpenClaw Agent.

## Goals

This repository packages CRE acquisition workflows as OpenClaw-native skills, workflows, workpads, schemas, and a deterministic local CLI.

## Non-goals

- No investment, legal, tax, accounting, brokerage, lending, or fiduciary advice.
- No real private deal data in fixtures.
- No silent external actions.
- No weakening of approval gates or disclaimers.

## Development

```bash
npm install
npm run lint
npm test
npm run validate
npm run sample
```

## Pull request checklist

- [ ] Tests added or updated.
- [ ] Docs updated when behavior changes.
- [ ] No secrets, tokens, credentials, or private deal materials committed.
- [ ] License and NOTICE attribution preserved.
- [ ] Security/privacy impact considered.
- [ ] Financial/legal/investment disclaimers not weakened.
- [ ] Synthetic fixtures only.

## Attribution

Where this project adapts concepts, architecture, workflow patterns, prompts, schemas, or documentation from CRE Acquisition Orchestrator, preserve attribution in `NOTICE` and relevant docs.
